## Get Started

### Start

- Copy `.env.exmaple` file and update with your setting

```sh
cp .env.exmaple .env
```

- Run docker-compose to watch and compile code

```sh
docker-compose up
```

- Run Apify

```sh
docker-compose exec node yarn apify
```

or watch

```sh
docker-compose exec node yarn start:watch
```
